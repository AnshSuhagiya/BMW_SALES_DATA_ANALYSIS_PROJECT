# BMW Sales Analysis 🚗📊

This is a beginner-friendly data analysis project focused on exploring global sales trends of BMW.

## 📌 Project Overview
The goal of this project is to understand sales patterns using data analysis techniques. It includes data cleaning, exploratory data analysis (EDA), and visualization.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Sales trend visualization
- Insights generation

## 📁 Dataset
The dataset used in this project is for educational purposes only.

## ⚠️ Disclaimer
This project is not affiliated with BMW. All data is used for learning and demonstration purposes only.

## 🚀 How to Run
1. Clone the repository
2. Open the notebook in Jupyter
3. Run all cells
